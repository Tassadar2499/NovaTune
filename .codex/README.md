# `.codex/`

Repository-local configuration and notes for Codex CLI runs.

This directory is intentionally tracked (via `.gitkeep`) so you can add Codex-related files here in future (e.g., runbooks, prompt snippets, or local agent configuration) without cluttering the repo root.

## Local Skills

Repo-specific Codex skills live under `.codex/skills/`.

These are the NovaTune-local imports of the original `.claude/skills/*` prompts, converted into Codex `SKILL.md` format for repository-scoped use.
